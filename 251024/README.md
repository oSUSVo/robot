포인터
- 메모리 주소를 저장하는 변수
- 변수는 값을 저장하며, 포인터는 그 값이 저장된 위치를 저장

- Code
    int a = 10;   // 변수 a에 10이라는 값이 저장됨
    int *p;       // 포인터 p 선언 (int형 변수를 가리키는 포인터)
    p = &a;       // p에 a의 주소를 저장
    
    이름	저장 내용	    의미
    a	    10	           값 자체
    &a	    0x7ffee0...	   a의 주소
    p	    0x7ffee0...	   a의 주소를 저장
    *p	    10	           p가 가리키는 주소에 저장된 값 (= a의 값)
